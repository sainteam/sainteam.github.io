S Λ I N T Ξ Λ M project

русскоязычное сообщество для поддержки игровых серверов на движке Source. Основные игры проекта, это Garry's Mod, CS 1.6, CS Source, CS:GO. Так же у проекта есть подраздел Отряд "Дельта", который занимается продвижением основного проекта среди русскоязычных серверов.

История:

Уже в конце лета (2017 года), была идея создания проекта SAINTEAM. Реализовался этот план уже в августе 18 числа. Сначало в проекте было четверо людей, 23 августа количество сотрудников увеличилось. Основателями проекта являются 4 друга, это Владислав Профет, Филипп Савинов, Вита Тихомирова и Орхан Четин, первые два основателя покинули проект по личным причинам.
